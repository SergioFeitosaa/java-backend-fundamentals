Java Fundamentals — Lógica de Programação e Estruturas de Controle

Contexto

Durante o desenvolvimento do meu primeiro projeto em Spring Boot, incluindo a implementação de um CRUD completo e a construção de regras de negócio, identifiquei a necessidade de reforçar minha base em lógica de programação e fundamentos de orientação a objetos.
Este repositório documenta esse processo de consolidação técnica.
O objetivo não é apenas resolver exercícios, mas desenvolver clareza lógica, controle de fluxo preciso e escrita de código mais consistente.

🎯 Objetivos Técnicos

Consolidar domínio sobre estruturas de controle em Java
Fortalecer raciocínio algorítmico
Melhorar organização e legibilidade de código
Evitar erros comuns de fluxo e lógica
Construir base sólida para aplicações backend com Spring Boot

🧩 Abordagem Utilizada
Cada exercício foi resolvido com foco em:
Entendimento do fluxo antes da implementação
Análise de possíveis erros lógicos
Testes manuais de comportamento
Comparação entre diferentes formas de resolver o mesmo problema
Otimização de loops quando possível
Exemplo de melhoria aplicada:
Substituição de verificação com % por incremento inteligente (i += 3) quando aplicável, reduzindo verificações desnecessárias.

🚀 Impacto no Desenvolvimento Backend
O reforço desses fundamentos contribui diretamente para:
Escrita de regras de negócio mais claras
Melhor organização em camadas (Controller, Service)
Maior segurança na manipulação de dados
Redução de bugs relacionados a fluxo
Melhor entendimento de estruturas usadas em APIs REST
Base forte → Código mais previsível → Desenvolvimento mais seguro.
