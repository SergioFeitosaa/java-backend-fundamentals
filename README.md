# 🚀 Java Fundamentals | Backend Development Journey

Bem-vindo ao meu repositório de estudos em **Java Backend**.
Este projeto documenta minha evolução prática no desenvolvimento, com foco na construção de uma base sólida em programação.

---

## 📌 Sobre o projeto

Este repositório foi criado para consolidar conhecimentos essenciais em Java, incluindo:

* Lógica de programação
* Programação Orientada a Objetos (POO)
* Métodos e organização de código
* Tratamento de exceções
* Boas práticas de desenvolvimento
* Bem como outros assuntos importantes que faz parte do dia a dia do desenvolvedor backend.

Todos os conteúdos aqui foram desenvolvidos com foco em **aprendizado prático**, simulando situações reais do dia a dia de um desenvolvedor backend.

---

## 📂 Estrutura do projeto

O projeto está organizado de forma progressiva, acompanhando minha evolução:

* `fundamentos/` → lógica, métodos e estrutura básica
* `poo/` → aplicação de conceitos orientados a objetos
* `desafios/` → resolução de problemas práticos
* `intermediario/` → evolução da complexidade dos exercícios

---

## 📈 Evolução contínua

Este repositório é atualizado constantemente com novos exercícios e desafios conforme avanço nos estudos.

Cada implementação representa um passo na minha jornada para me tornar um desenvolvedor backend mais preparado e consistente.
